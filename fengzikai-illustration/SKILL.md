# 丰子恺水墨插图 / Feng Zikai Ink-Wash Illustration

## 核心定位

把文章观点、生活场景、哲理思考和内容资产，转译成一张有温度的丰子恺水墨风格配图：

```
简笔人物 + 水墨淡彩 + 大面积留白 + 手写题字 + 生活诗意
```

这不是技术信息图，也不是商业插画。目标是让读者先感受到一种安静、温暖、有诗意的画面，再在一秒内被题字点出文章的核心情绪。

## 丰子恺风格 DNA

丰子恺（1898-1975），中国现代漫画先驱。他的画有以下核心特征：

### 视觉特征
- **水墨线条**：用软毛笔勾勒，线条有粗细变化、起笔收笔有顿挫，不是均匀的硬笔线条
- **淡彩着色**：在上好墨线后，用极淡的水彩或国画颜料薄薄罩一层，颜色不饱和、不跳脱——淡赭石、花青、藤黄、淡朱砂
- **简笔人物**：人物没有五官细节，没有解剖结构，用三五笔勾出姿态和神韵。衣服常用淡色平涂或留白
- **大面积留白**：画面 50%-70% 是空白（模拟宣纸），主体集中在画面中下部或一侧。留白不是"没画完"，而是构图的一部分
- **宣纸质感**：背景是暖白/米白，带有极轻微的纸纤维纹理，不是纯数字白 `#FFFFFF`
- **手写题字**：画面上方或一侧有竖排或横排的手写毛笔题字，内容是一句诗、一句哲理或画题。字体质朴自然，不是印刷体
- **红色印章**：通常在题字下方或画面角落有一方小小的朱红色印章（1-2 个）
- **淡墨渲染**：远山、树影、地面常用淡墨或淡彩大笔扫出，边界模糊

### 内容特征
- **日常生活**：喝茶、看花、读书、聊天、散步、带孩子、看月亮、听风
- **儿童天真**：放风筝、捉蝴蝶、爬树、玩水、逗猫
- **自然诗意**：柳树、燕子、月亮、山影、江水、小舟、梅花、竹子
- **人间温情**：母子、师生、朋友、邻里，人与人之间有含蓄的情感交流
- **哲理禅意**：画面安静，常常有一句引人思考的题字，"人散后一钩新月天如水""红了樱桃绿了芭蕉"
- **民国风物**：长衫、折扇、茶壶、竹椅、纸伞、灯笼、石板路、木窗

### 不要的画面元素
- 没有现代电子产品（电脑、手机、耳机）
- 没有西方物件（咖啡杯、西装、汽车、高楼）
- 没有饱和鲜艳的颜色
- 没有复杂背景和过多细节
- 没有写实人像和五官
- 没有对话框、气泡、信息图元素
- 没有几何图形、图表、箭头

## Before Starting

按任务需要读取，不要一次全部塞进上下文：

- `references/style-dna.md`：丰子恺视觉 DNA、比例、留白、色彩、线条、题字规则
- `references/prompt-template.md`：生图提示词模板（含正向和负向约束）
- `references/story-extraction.md`：如何从正文提炼场景、人物和题字
- `references/master-selection.md`：参考母版选择、变异要求、出图拦截规则
- `references/qa-checklist.md`：生成后质量检查、失败信号和迭代方向

开始前确认：
- 用户是否明确要单张配图还是系列配图
- 文章的主题情绪是什么（温暖/哲思/童趣/怀旧/禅意/日常）
- 用户是否提供了必须出现的特定场景或人物
- 如果没有明确要求生成图片，只先输出 shot list 或配图方案

## Core Flow

### 1. 消化内容

先提炼当前内容里的"情绪"和"场景"，不要急着想画面：

- 这段文字的核心情绪是什么？温暖？怀念？孤独？喜悦？宁静？哲思？
- 有没有一个具体的日常场景可以承载这个情绪？
- 丰子恺画过的场景里，哪一个最接近？（喝茶、望月、散步、读书、柳下、舟中、庭前、窗边……）
- 哪一句可以作为题字？不能是"金句"或"slogan"，必须是自然的中文短句，最好有诗感
- 是否有自然元素可以入画？（柳、燕、月、山、水、花、竹、石）

完成标记：已经明确核心情绪、日常场景、主要人物关系、题字内容和自然元素。

### 2. 先出 shot list

如果用户说"分析 / 思考 / 怎么配图"，先给 shot list。每张写清：

- 场景描述（日常生活的哪个切片）
- 人物关系（单人/两人/多人，在做什么）
- 情绪基调
- 核心自然元素
- 题字内容（一句）
- 构图方向（竖/横/方，主体在画面哪儿）
- 为什么这张值得画（和文章观点的关联）

默认 1-3 张备选。用户明确要"画出来 / 生成 / 出图"时，直接进入生成流程。

完成标记：每个候选画面都有场景、人物、情绪、元素、题字和成图理由。

### 3. 母版锁定

生成前必须完成母版锁定。读取 `references/master-selection.md`，从 01-06 中选一张最匹配的质量锚点。

必填字段：

```
母版：
抽取的不变量：
当前内容的变异点：
画面情绪句：
当前内容适配：
要避免的失败信号：
```

母版不是可复制构图。必须至少 3 项和母版不同：场景、人物数量/姿态、自然元素、题字内容/位置、季节/时间。

完成标记：每张图都有明确母版、不变量、至少 3 个变异点、情绪句和失败信号。

### 4. 单张生成

每张图单独生成，不要把多张拼成一张。

默认用竖版 3:4 或 2:3（丰子恺原作以竖幅和方幅为主，横版适合广阔场景）。

提示词必须包含：
- 丰子恺风格触发词：`FENG ZIKAI style`
- 水墨线条 + 淡彩着色
- 宣纸质感背景，暖白/米白
- 简笔人物（无五官细节）
- 大面积留白（50%-70%）
- 手写毛笔题字（具体题字内容）
- 红色印章（1-2 个）
- 具体场景和自然元素
- 负面约束（禁止项列表）

完整模板见 `references/prompt-template.md`。

完成标记：每张图都有独立提示词，包含风格触发词、场景、人物、题字、色彩、留白和负向约束。

### 5. 生成后 QA

第一张生成图只算候选，不算交付。交付前必须对照 `references/qa-checklist.md` 逐项检查：

- 候选图是否有水墨线条感（不是数字硬笔）？
- 色彩是否淡雅（不是饱和鲜艳）？
- 留白是否足够（不是满版）？
- 人物是否简笔（不是写实有五官）？
- 题字是否像手写（不是印刷体）？
- 是否有红色印章？
- 背景是否有宣纸质感？
- 画面是否 3 秒传达情绪？

任一项明显失败，不能交付为最终图，必须重写提示词或重生成。

## 风格边界

### 丰子恺风格 ≠ 
- 日本浮世绘（线条更细密、色彩更平面）
- 水墨山水画（太宏大、太满、没有人物生活）
- 国画工笔（太精细写实）
- 新中式插画（太设计感、太几何化）
- 儿童绘本（太卡通、太鲜艳）
- 禅绕画（太抽象、太装饰）

### 丰子恺风格 ≈
- 一个懂生活的人，用毛笔画下了今天下午喝茶时的样子
- 不是"画画"，是"记事"
- 画面的主角不是技法，是生活本身
