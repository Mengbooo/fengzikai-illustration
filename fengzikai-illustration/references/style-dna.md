# 风格 DNA

## 一句话

宣纸上的中国生活诗：用淡墨简笔画出日常的一个切片，大面积留白，一句手写题字道出画外之意。

## 核心公式

```
一个日常场景 + 简笔人物（无五官） + 淡彩着色 + 自然元素 + 大面积留白 + 手写毛笔题字 + 红色印章
```

## 画面比例

丰子恺原作以竖幅和方幅为主，适合表达纵向空间（柳条下垂、山影远淡、人物在下方活动）。横版适合表达广阔的自然场景（江水、远山、柳岸）。

### 竖版 3:4（最常用）

- 适合：读书、喝茶、窗前、树下、月下、两人对坐
- 主体在画面中下部（下方 1/3 到 1/2 处）
- 上方大面积留白，题字在右上或左上
- 自然元素（柳枝、竹叶、远山）从上方或侧方垂下/延伸

### 方版 1:1

- 适合：对称构图、庭院场景、多人场景
- 主体居中或偏下
- 留白均匀分布在主体四周
- 题字可在任意一侧

### 横版 16:9 或 4:3

- 适合：江水、远山、柳岸、舟行、多人散步
- 主体偏下 1/3 处
- 上方和左右留白
- 题字在右上或左上角

## 色彩系统

丰子恺用色极其克制。核心原则：**墨线为主，淡彩为辅**。

### 主色
- **墨黑** `#2C2C2C` → `#1A1A1A`：人物轮廓、主要线条、题字
- **淡墨灰** `#8A8A8A` → `#B0B0B0`：远山、树影、地面渲染
- **宣纸白** `#F5F0E8` → `#FAF6EE`：背景底色，带极轻微暖黄

### 淡彩（极淡、薄涂）
- **淡赭石** `rgba(180,130,90,0.3)`：人物衣服、树干、地面
- **花青** `rgba(80,120,160,0.25)`：远山、水色、衣纹
- **藤黄** `rgba(200,180,100,0.25)`：月亮、花蕊、光晕
- **淡朱砂** `rgba(190,100,90,0.4)`：印章（唯一较浓的颜色）、花朵点缀
- **淡石绿** `rgba(120,160,130,0.2)`：柳叶、草色、春意

### 色彩铁律
- 任何颜色的饱和度都不能高。没有"鲜红""亮黄""天蓝""翠绿"
- 淡彩是罩在墨线上的，不能盖住墨线
- 大面积色彩用大笔触，边界模糊，不要均匀平涂
- 一幅画最多出现 3 种淡彩色

## 线条规则

### 人物线条
- 毛笔软锋质感：起笔略顿、行笔流畅、收笔有提按
- 粗细变化自然：衣纹转折处略粗，肢体轮廓较细
- 3-5 笔一个人：头一笔（圆点），身体两笔（弧线），腿一笔（折线），手臂一笔
- 无五官：脸上是空白，或用淡墨在发际点一笔表示头发
- 人物姿态来自身体线条的弧度，不来自解剖结构

### 自然元素线条
- **柳枝**：细长弧线，从上方垂下，3-5 条就够了。线条要有弹性，不是死直线
- **山影**：淡墨横扫，上实下虚，边界模糊
- **树干**：略粗的断续线条，有飞白感
- **水纹**：极淡的横线，2-3 条，间距宽
- **竹叶**：短笔触，"个"字形，3-5 组

## 留白规则

留白不是"没画"，是构图的核心语言。

- 画面 50%-70% 是空白
- 主体不能居中填满，要偏下、偏左或偏右
- 主体和题字之间要有呼吸距离
- 留白区域有宣纸的微暖底色，不是纯白 `#FFFFFF`
- 物件之间要有空隙，不能挤在一起
- 地面通常不画满——一条淡墨横线或淡赭石色块就够了

## 题字规则

### 位置
- 竖排题字：画面右上角或左上角，从上往下写
- 横排题字：画面上方（偏左或偏右），一行
- 题字不压在人物头上，不嵌在主体中间

### 内容
- 一句诗、半句词、一句白话哲理
- 3-12 个字为佳
- 不要"金句""slogan""标题感"
- 好的题字："人散后，一钩新月天如水""红了樱桃，绿了芭蕉""春日游，杏花吹满头"
- 当代场景可以写白话："下午三点，茶凉了""窗外在下雨""猫睡着了"
- 题字必须和画面情绪一致，不能是解释画面的"说明书"

### 字体
- 手写毛笔行书或楷书，质朴自然
- 不要印刷体（宋体、黑体、美术字）
- 不要过分潦草的狂草
- 字的大小可以微微不齐，更有手写感

## 印章规则

- 1-2 个朱红色小方章
- 位置：题字下方、画面左下角或右下角
- 大小：占画面宽度的 1/20 到 1/15
- 颜色：朱砂红 `#C04040` → `#A03030`（不要太鲜艳）
- 印章内容：可以是白文或朱文，篆书风格，不必可读
- 不要太多印章，不要把画面变成印章收集页

## 禁止项总览

### 数字感
- 没有均匀线条、几何图形、图表、箭头
- 没有矢量扁平风格
- 没有渐变色块

### 现代感
- 没有电子产品（手机、电脑、耳机、平板）
- 没有现代服饰（西装、T恤、运动鞋、棒球帽）
- 没有现代建筑（玻璃幕墙、钢筋水泥、霓虹灯）
- 没有交通工具（汽车、飞机、地铁）

### 商业感
- 没有 Logo、品牌色、二维码
- 没有对话框、气泡
- 没有信息图模块、步骤编号
- 没有 PPT 排版感

### 饱和感
- 没有鲜艳颜色
- 没有纯黑纯白的大面积对比
- 没有暗角、光晕、滤镜效果
- 没有 3D 渲染质感
