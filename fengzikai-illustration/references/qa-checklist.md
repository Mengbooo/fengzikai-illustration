# QA Checklist

## 必过项

- [ ] 使用的是丰子恺水墨淡彩风格（不是数字扁平、不是日本浮世绘、不是儿童绘本）
- [ ] 背景是暖白宣纸色（接近 `#F5F0E8`），不是纯白 `#FFFFFF`
- [ ] 背景有轻微纸纤维纹理感，不是光滑数码白
- [ ] 线条有毛笔质感——粗细变化、起笔收笔有顿挫，不是均匀数字线条
- [ ] 人物简笔——3-5 笔勾勒，无五官细节
- [ ] 色彩极度克制——只有墨黑 + 2-3 种极淡的水彩色
- [ ] 饱和度极低——没有鲜艳的红/黄/蓝/绿
- [ ] 留白充分——画面 50%-70% 是空的
- [ ] 主体不居中填满——偏下、偏左或偏右
- [ ] 有手写毛笔题字——字体自然质朴，不是印刷体
- [ ] 题字内容 3-12 字，是诗句或白话哲理，不是标题
- [ ] 有 1-2 个朱红色小印章
- [ ] 没有现代物品——手机、电脑、耳机、汽车、西装、咖啡杯
- [ ] 没有商业元素——Logo、QR码、对话框、信息图模块
- [ ] 画面 3 秒能传达一个明确的情绪
- [ ] 整体质量达到丰子恺原作的精神——安静、温暖、有诗意

## 生成后逐项比对

第一张生成图只算候选，不算交付。交付前必须逐项判断：

### 线条质量
- [ ] 候选图的线条是毛笔质感还是数字硬笔？
- [ ] 线条有自然的粗细变化吗？
- [ ] 人物是 3-5 笔简笔还是写实素描？

### 色彩
- [ ] 候选图的色彩是否饱和度极低？
- [ ] 是否有任何一眼就"跳出来"的鲜艳颜色？
- [ ] 淡彩是薄涂罩染感还是平涂色块？
- [ ] 背景是暖白宣纸色还是纯白 `#FFFFFF`？

### 留白
- [ ] 留白是否达到 50%-70%？
- [ ] 留白区域是否有宣纸质感还是纯白空？
- [ ] 主体是否挤在一起还是呼吸充分？

### 人物
- [ ] 人物是否有五官？（有 → 失败）
- [ ] 人物姿态是否能传达情绪？（不能 → 重写动作）
- [ ] 人物是否像丰子恺笔下的简笔小人？（不像 → 失败）

### 题字
- [ ] 题字是否像手写毛笔字？（不像 → 失败）
- [ ] 题字内容是否有诗意而不是标题感？
- [ ] 题字位置是否自然不突兀？

### 印章
- [ ] 是否有红色印章？
- [ ] 印章是否过大或过艳？

### 情绪传达
- [ ] 不看题字，3 秒能感受到情绪吗？
- [ ] 情绪和文章匹配吗？

### 母版比对
- [ ] 候选图是否像所选母版的换皮版？（像 → 失败）
- [ ] 候选图至少有 3 个明显变异吗？

## 失败信号

出现以下情况，重写提示词或重生成：

### 风格失败
- 变成数字扁平风（SVG 感、矢量感、均匀线条）
- 变成日本浮世绘（更细密的排线、平面色块）
- 变成儿童绘本（太卡通、太鲜艳、五官明确）
- 变成写实水墨（太精细、太满、太"像照片"）
- 变成新中式插画（太设计感、几何化）
- 变成钢笔速写（硬笔线条，无毛笔质感）

### 色彩失败
- 任何饱和鲜艳的颜色
- 背景是纯白 `#FFFFFF`
- 墨色太淡，画面灰蒙蒙；或太深，像版画
- 淡彩变成不透明平涂
- 用了超过 3 种淡彩色

### 构图失败
- 留白不足 40%
- 主体居中且填满
- 元素超过 5 种（堆砌）
- 题字压住人物

### 内容失败
- 出现现代物品
- 人物有五官
- 题字是印刷体
- 没有印章
- 画面"空"但没有意境——只是没画完

### 情绪失败
- 画面没有任何情绪（只是"有一个人在喝茶"）
- 情绪和文章完全对不上
- 3 秒读不懂任何信息

## 比例自查

问自己：
- 我现在选的是哪一张母版？候选图像它的"同族图"吗？
- 候选图有没有第一眼像母版换皮？如果有，判失败
- 留白够吗？有没有 50% 以上是空白的宣纸？
- 如果有人把这幅画截屏发朋友圈，别人会认出是"丰子恺风格"吗？

## 修复指南

| 问题 | 修复方向 |
|------|----------|
| 线条太硬 | 加重"soft calligraphy brush""visible variation in line thickness" |
| 颜色太艳 | 加重"extremely muted""transparent wash""barely-there color" |
| 留白不够 | 加重"50-70% empty space""generous negative space" |
| 背景太白 | 加重"warm off-white xuan paper""#F5F0E8""NOT pure white" |
| 人物太写实 | 加重"3-5 brush strokes""no facial features""blank face" |
| 题字太印刷 | 加重"hand-brushed""naturally irregular""NOT typeset" |
| 缺印章 | 明确加"1-2 small square vermilion seal marks" |
| 现代物品 | 加重"NO electronic devices""NO modern objects" |
| 堆砌元素 | 减少场景元素数量到 1-3 个 |
| 母版复刻 | 增加变异要求，改变场景/时间/人物/元素 |
