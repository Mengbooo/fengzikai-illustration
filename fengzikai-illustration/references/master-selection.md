# 母版选择与拦截协议

本文件用于修复两个关键风险：
- **元素堆砌**：模型把场景、人物、自然元素当成清单全部堆进画面
- **母版复刻**：模型把参考母版当成描摹模板，直接复制构图

正确做法是先选择一个母版作为**质量锚点**，再抽取它的抽象不变量，最后为当前内容写出必须变异的地方。母版不是可复制画面；它只提供质量标尺和情绪类型。

## 母版使用总原则

每次使用母版必须同时满足：
- **保留不变量**：比例克制、留白充分、简笔人物、淡彩薄涂、题字自然
- **必须变异**：场景主题、人物数量/关系、自然元素、题字内容/位置、季节/时间中，至少 3 项必须和母版不同
- **情绪先行**：不看题字，3 秒内也能感受到画面的情绪基调

如果生成图第一眼像某张母版的换皮版，判失败。

## 标准母版索引

### 01 tea-and-conversation

参考母版：
```
Pinterest 丰子恺茶画：两人对坐喝茶，窗外有景，桌面简单一壶两杯
```

适合：
- 对话、交流、分享、朋友相聚、师徒传授、访谈感

抽象骨架：
- 两人对坐（一左一右或一近一远）
- 中间有桌面或茶几
- 上方或侧方有自然元素探入
- 一人执壶/举杯，一人倾听/等待
- 画面安静，有"说与听"的关系

可复用关系：
- "两人 + 桌面 + 窗外景"的空间结构
- 一主一客的人物关系
- 茶壶/茶杯作为人物之间的连接物

变异要求：
- 不能照搬"左窗 + 中桌 + 两人对坐"
- 可以：一人坐一人站、桌上无茶具而有书、窗变门、室内变室外
- 至少改变：场景（茶室→庭院/柳下）、人物关系（对坐→并立）、题字内容

失败信号：
- 直接复制"左窗 + 右桌 + 两人对坐喝茶"
- 茶具变成咖啡杯或西式茶具
- 窗外是现代城市景观

### 02 moon-and-solitude

参考母版：
```
丰子恺望月图：一人倚栏或凭窗望月，柳枝垂下，远山淡影，大面留白
```

适合：
- 孤独、思念、独处、深夜、哲思、怀远

抽象骨架：
- 一个人物，侧影或背影
- 人物在画面中下部，面朝上方或远方
- 上方有月亮
- 柳枝或树枝从上方垂下
- 远山淡影在远景

可复用关系：
- "一人 + 月 + 柳"的经典组合
- 人物面朝画外（月亮方向），不面对读者
- 大量留白表达"空"和"远"

变异要求：
- 不能照搬"左下人物 + 右上月亮 + 上方柳枝"
- 可以：白天非月夜（改成看花、看山、看远方）、人物姿态变化（坐/立/靠）、季节变化
- 至少改变：时间（夜→黄昏）、人物姿态、自然元素（柳→竹/松）

失败信号：
- 月亮太亮太大，像摄影作品
- 人物有表情细节
- 背景填满，没有留白的"空"感

### 03 children-playing

参考母版：
```
丰子恺儿童图：小孩在户外玩耍，放风筝、捉蝴蝶、爬树、追跑
```

适合：
- 童趣、天真、快乐、教育、成长、亲子

抽象骨架：
- 1-3 个儿童
- 在户外自然环境中
- 有一个明确的游戏动作
- 画面有动感但不凌乱
- 儿童身形矮小，肢体线条圆润

可复用关系：
- "儿童 + 自然 + 玩耍动作"
- 儿童矮小的比例关系（和树、石等对比）

变异要求：
- 不能照搬"放风筝"或"捉蝴蝶"
- 可以：玩水、爬树、看蚂蚁、追猫、摘花
- 至少改变：游戏内容、季节、人数、自然元素

失败信号：
- 儿童有五官和表情
- 变成卡通风格
- 颜色太鲜艳像儿童绘本

### 04 spring-and-nature

参考母版：
```
丰子恺春景图：柳条轻垂、燕子归来、远山淡影、人在自然中很小
```

适合：
- 春天、生机、希望、复苏、自然之美

抽象骨架：
- 自然元素占画面主导（60%+）
- 人物很小，融在自然里
- 柳枝、燕子、花是核心元素
- 画面轻快，淡彩以花青和藤黄为主

可复用关系：
- "大自然 + 小人物"
- 上方垂柳 + 空中飞燕 + 远山淡影

变异要求：
- 不能照搬"上方柳 + 空中燕 + 下方小人"
- 可以：换季节（秋菊、冬梅）、换动作（散步→坐看）、换视角
- 至少改变：季节、主要自然元素、人物在画面中的位置

失败信号：
- 自然元素种类太多（超过 3 种）
- 人物变大，变成"人在景前合影"
- 燕子画得像写实鸟类图鉴

### 05 reading-and-quiet

参考母版：
```
丰子恺读书图：一人独坐读书，书桌前、窗边或树下
```

适合：
- 安静、学习、读书、内省、专注、一个人的时光

抽象骨架：
- 单人
- 坐着（椅/石/台阶/树根）
- 手中或面前有书
- 环境安静，自然元素在旁不喧宾夺主
- 窗或树作为空间界定

可复用关系：
- "一人 + 书 + 安静空间"
- 侧光或顶光的感觉（通过淡墨表达）

变异要求：
- 不能照搬"右窗 + 左人 + 中桌 + 书"
- 可以：树下读书、石上读书、灯下读书、舟中读书
- 至少改变：环境（室内→室外）、时间（白天→夜晚灯下）、人物姿态

失败信号：
- 书是现代精装本或 Kindle
- 环境太现代（办公室、咖啡馆）
- 人物有眼镜、耳机等现代配件

---

### 06 walking-and-paths

参考母版：
```
丰子恺山径图：一或两人在山间小路上行走，远山淡影
```

适合：
- 人生、旅程、陪伴、前行、离别、重逢

抽象骨架：
- 一条蜿蜒小路
- 1-2 人在路上
- 远山或树影作为景深
- 人物小，路径长，空间感强

可复用关系：
- "路 + 人 + 远山"的纵深感
- 人物背影或侧影，面朝远方

变异要求：
- 不能照搬"山径 + 两人背影 + 远山"
- 可以：水边路、林间路、一人独行、两人并肩
- 至少改变：路面环境、人物关系、方向（左→右或右→左）

失败信号：
- 路太规整像公路
- 远山画得太实，没有"远"的空气感
- 人物太多（超过 3 个）

## 母版锁定模板

每次选定母版后必须填写：

```
母版：{01-06 之一}
抽取的不变量：{比例/留白/简笔人物/淡彩/题字/情绪等}
当前内容的变异点（至少 3 个）：
  1. {变异点1，如：场景从茶室改为柳下}
  2. {变异点2，如：人物从对坐改为并立}
  3. {变异点3，如：季节从春改为秋}
画面情绪句：{不看题字，3 秒感受到什么}
当前内容适配：{为什么这个母版适合这段内容}
要避免的失败信号：{最容易犯的 2-3 个错误}
```
