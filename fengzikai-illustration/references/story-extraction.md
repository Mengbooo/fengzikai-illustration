# 故事提炼

## 目标

从一段文字里提炼出一个可以用丰子恺风格表达的具体日常场景。不是画文章结构，是画文章情绪。

## 五步提炼法

### 1. 找核心情绪

先不想要画什么，先想这篇文章让人感受到什么：

- **温暖**：人与人之间的关怀、陪伴、默契
- **怀念**：对过去的回望、对旧时光的温柔
- **童趣**：天真、好奇、简单的快乐
- **哲思**：安静中的顿悟、生活的道理
- **孤独**：一个人的时刻，但不悲伤，是"独处"
- **宁静**：放下、平和、岁月静好
- **诗意**：美在日常、不经意间的感动
- **禅意**：空、淡、无、简

如果一篇文章同时有多个情绪，选最主要的一个。一幅画只能承载一种情绪。

### 2. 找到日常切片

情绪需要落到一个具体的日常场景。问自己：

- 这个情绪在什么日常时刻自然出现？
- 丰子恺画过的 30 个经典场景里，哪个最接近？

丰子恺常用场景清单：
- 喝茶、饮酒、吃饭
- 看书、写字、画画
- 望月、看花、听风、赏雨
- 散步、闲坐、倚栏、凭窗
- 柳下、竹旁、梅边、松间
- 舟中、桥上、庭前、山径
- 逗猫、戏儿、聊天、下棋
- 送别、重逢、等待、思念
- 放风筝、捉蝴蝶、爬树、玩水
- 母抱子、师授徒、友对酌、邻相语

如果内容对应不上任何传统场景，创造一个当代但仍"中国生活"的场景。关键规则：
- 人在自然中（有树/花/山/水/月）
- 动作简单安静（不是运动、不是忙碌）
- 有时间感（午后/黄昏/月夜/春日/秋日）

### 3. 设计人物关系

丰子恺的人物不靠表情交流，靠身体姿态：

- **单人**：背影、侧影最多。人在看什么（看花、看月、看远方），给读者代入感
- **两人**：对坐、并立、一前一后。距离近但不碰触，有默契感。一人说话一人听，一人斟茶一人等
- **三人及以上**：疏密有致，不能排成一行。2+1 组合最常见

人物姿态传递情绪：
- 微微低头 = 沉思、安静
- 抬头仰面 = 望月、感慨
- 身体微侧 = 倾听、关注
- 手托腮 = 闲适、发呆
- 双手背在身后 = 散步、从容
- 怀抱 = 温暖、呵护

### 4. 挑选自然元素

自然元素不是"背景"，是和人物共同构成意境的"角色"。一幅画选 1-3 个自然元素：

常见的元素-情绪配对：
| 元素 | 情绪 |
|------|------|
| 柳枝 | 春意、离别、温柔 |
| 月亮 | 思念、孤独、永恒 |
| 燕子 | 春天、归来、生机 |
| 梅花 | 清高、冬意、坚韧 |
| 竹子 | 气节、清瘦、简洁 |
| 远山 | 淡远、超脱、时间 |
| 江水 | 流逝、旅程、开阔 |
| 小舟 | 漂泊、独处、自由 |
| 芭蕉 | 雨声、闲适、南国 |
| 菊花 | 秋意、隐逸、淡泊 |
| 松树 | 苍劲、老友、长久 |
| 桃花 | 春色、美好、短暂 |

不要把所有想到的元素都放进去。丰子恺的画是减法。

### 5. 写题字

题字是画外之音。不是标题，不是说明。

**好的题字**：
- "人散后，一钩新月天如水"（画几个人散去，剩一把空椅子，天上新月）
- "红了樱桃，绿了芭蕉"（画窗外芭蕉，桌上樱桃）
- "儿童不知春，问草何故绿"（画小孩蹲在地上看草）
- "今夜故人来不来"（画一个人倚门望月）
- "山路元无雨，空翠湿人衣"（画山间小径，满目苍翠）

**当代白话题字**（如果文章是现代内容）：
- "下午三点，茶凉了"
- "猫比我懂生活"
- "窗外的树比昨天绿"
- "看完这一页就去睡"
- "妈妈在厨房，灯是暖的"

**题字禁忌**：
- 不要"金句"感："格局打开""认知升级""底层逻辑"
- 不要Slogan："让生活更美好""遇见更好的自己"
- 不要说明文："图为一个孤独的人在看月亮"
- 不要网络用语："笑死""绝了""离谱"
- 不要英文

### 输出格式

提炼完成后，输出如下：

```
情绪：{核心情绪}
场景：{日常切片，一句话}
人物：{几人，什么关系，在做什么}
自然元素：{1-3 个}
题字："{一句}"
构图：{竖/方/横}
淡彩色：{用了哪 2-3 种}
```
