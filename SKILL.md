# 丰子恺水墨插图

## 核心定位

用丰子恺的方式画画——不是画"技法"，是画"生活里那个有趣的瞬间"：

```
毛笔线条 + 简笔人物 + 留白 + 题字 + 生活趣味
```

丰子恺的画有三个容易被忽略的特质：**温暖、俏皮、不严肃**。他画小孩打架、猫咪捣乱、大人出糗，都带着一种"觉得好笑但不说破"的温柔。不要让画面变成端庄的"国风插画"——那恰恰是丰子恺最不想要的。

---

## 丰子恺风格 DNA

### 线条——毛笔的"活"感

这不是日本漫画的均匀硬线，也不是书法作品的精雕细琢。丰子恺的线条：
- **有快慢节奏**：快速扫过的弧线（衣纹、柳条）和缓慢顿挫的短笔（树干、石纹）交替出现
- **墨色有浓淡**：同一幅画里，人物轮廓可能是浓墨，远山和地面是淡墨，层次靠墨的含水量而非颜色
- **笔触可见**：你能看出毛笔的"飞白"（笔锋分叉露出的纸底）、起笔的圆点、收笔的尖尾
- **不追求"准"**：线条偶尔画过头、画歪了、接头没对齐——这是手绘感，不是 bug

### 题字——是"随手写的"，不是"书法作品"

这一点尤其重要。丰子恺的题字字体：
- **质朴行书**：介于楷书和行书之间，偏向日常书写，不是展览级书法
- **字大小不齐**：第一个字可能比第三个字大一圈，竖排时字间距不完全均匀
- **笔画省略**：某些字少一笔，或者连笔过于随意——像朋友在便签上留言
- **墨色和画面一致**：题字用的墨和画画的墨是同一种，不是印刷黑
- **位置随意**：可能歪在右上角，可能挤在柳枝旁边，不是在 Photoshop 里对齐过的

**不要**：
- 不要碑帖级别的楷书（太正式）
- 不要狂草（太潦草、和画面脱节）
- 不要印刷体/美术字（致命伤）
- 不要把字排得像排版软件对齐的

### 人物——三笔五笔的"神"

丰子恺的人物不是"简化"，是"提炼"——用最少的笔画抓住最有神采的姿态：
- 头 = 一个墨点或小圆圈，有时歪着（小孩抬头看鸟）
- 身体 = 一两笔弧线，肩膀的圆润或驼背通过弧线弧度表达
- 四肢 = 单线，但有动作感——手向上够和手下垂，线的角度完全不同
- **没有五官**，但你能看出情绪——靠头的角度、身体的倾斜、手的动作

### 色彩

两种模式可选：

**淡彩模式（默认）**：
- 墨线是骨架，彩是"随手罩一层"
- 淡赭石（衣）、花青（山/水）、藤黄（月/花蕊）、极淡朱砂（印章/花点）
- 任何颜色饱和度都极低——像在水里涮了三遍的毛笔轻轻扫过

**纯黑白模式**：
- 只有墨色，没有其他颜色
- 靠墨的浓淡干湿区分层次：浓墨（人物、前景线）、中墨（树干、石）、淡墨（远山、水纹、地面渲染）
- 印章可以保留朱红色（黑白画里唯一的颜色，丰子恺原作常见）
- 如果用户说"黑白""水墨""纯墨色""不要颜色"，启用此模式

### 留白——不是没画，是"留给读者想"

画面 40%-60% 是空白的宣纸。留白不是构图事故，是丰子恺最核心的语言——他在《缘缘堂随笔》里说"空白不是无，是余韵"。

### 宣纸底色

背景是暖白/米白，带极轻微纸纤维感。不是纯白 `#FFFFFF`。

## 内容——不要限制想象力

丰子恺画过：小孩对着墙上自己的影子敬礼、两只猫打架掉下屋檐、一个人爬到树上躲雨、大人对着锅里煮的螃蟹一脸无辜。他的题材**不限于"古典温婉"**——

**可以画**：
- 日常诗意：喝茶、望月、散步、读书、聊天
- 儿童趣事：打架、爬树、放风筝、逗猫、对着蚂蚁发呆
- **生活中的滑稽瞬间**：踩到水坑、伞被风吹翻、被猫绊了一跤、写字时墨甩到脸上
- 自然：柳、燕、月、山、水、花、竹、芭蕉
- 人间关系：母抱子、师授徒、朋友对酌、邻居聊天
- **当代生活转译**：用丰子恺的笔法画当代的日常——但要"翻译"成民国风物（手机不出现，但"低头看东西"的姿态可以保留，换成看书/看信）

**不要的只是视觉元素**（这些东西会破坏风格的统一性）：
- 电子产品、现代交通工具、现代建筑外观
- 饱和度高的鲜艳颜色
- 写实人像和五官细节
- 对话框、信息图、商业排版

## 两种模式

### 淡彩模式（默认）

完整使用淡赭石、花青、藤黄等极淡色。适合表达温暖、春天、生活气息。

### 纯黑白模式

触发词：黑白、水墨单色、纯墨色、不要颜色、墨色

仅用墨的浓淡层次。适合表达哲思、寂静、怀旧、冬意。印章保留朱红色作为唯一彩色。

## Core Flow

### 1. 消化内容

先找内容的情绪，再找一个"画面瞬间"而非"主题概念"：

- 核心情绪是什么？（温暖？俏皮？怀念？宁静？滑稽？）
- 有没有一个**具体的生活瞬间**可以承载？（注意：是"瞬间"，不是"概念"。不是"沟通的重要性"，而是"两个人说话，一个人把茶洒了"）
- 题字怎么写才有"随手题"的感觉？
- 用淡彩还是纯黑白？

### 2. Shot list → 生成 → QA

流程同母版系统，见 `references/` 文件。
