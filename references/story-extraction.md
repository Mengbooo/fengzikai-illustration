# 故事提炼

## 目标

从一段文字里找到一个**小孩都能画出来的瞬间**。

---

## 三步法

### 1. 找"一个瞬间"

不是找主题，是找**画面里正在发生的那一秒**：

- ✅ "一个人踩在凳子上，伸手够枇杷"
- ✅ "两只猫打架，一只从屋檐掉下来"
- ✅ "下雨了，一个人拿荷叶挡雨"
- ✅ "两个小孩蹲着看蚂蚁"
- ❌ "人与自然的和谐"
- ❌ "童年的快乐"
- ❌ "沟通的重要性"

### 2. 拆成"一笔画得完"的元素

丰子恺的画最多 3-4 个元素：

| 元素 | 怎么画 |
|------|--------|
| 人物 | 一个圈 + 几条线 |
| 树 | 一条竖线 + 一个圈 |
| 桌子 | 一个方框 + 两条线 |
| 茶壶 | 一个圈 + 一个嘴 + 一个把 |
| 月亮 | 一个不圆的圈 |
| 山 | 一条歪弧线 |
| 地面 | 一条横线 |

如果一张画里元素超过 4 个，说明太复杂了——删。

### 3. 写题字

小孩想到什么写什么。简单、直接、不讲究：

- "枇杷熟了"
- "猫打架"
- "下雨了"
- "他还在等"
- "茶凉了"

---

## 输出格式

```
模式：{淡彩 / 纯黑白}
瞬间：{一句话，10 字内}
人物：{几人 + 简单动作}
物件：{最多 1 个}
题字："{3-8 字}"
```
