# 故事提炼

## 目标

从一段文字里找到一个"可画的瞬间"。不是画文章结构，是画文章里最有生活感的那一秒。

---

## 五步提炼法

### 1. 找核心情绪

丰子恺的情绪光谱比一般人想的宽：

| 情绪 | 关键词 |
|------|--------|
| 温暖 | 陪伴、关怀、默契、日常 |
| 俏皮 | 滑稽、出糗、好笑但不刻薄 |
| 童趣 | 天真、好奇、认真做小事 |
| 怀念 | 旧时光、回不去、温柔 |
| 哲思 | 安静中的顿悟、生活的道理 |
| 宁静 | 独处、放下、岁月静好 |
| 诗意 | 美在日常、不经意间 |
| 滑稽 | 踩水坑、伞吹翻、猫捣乱——生活的小笑话 |

### 2. 找到"瞬间"——不是"概念"

这是最关键的一步。问自己：如果丰子恺看到这段文字，他会画里面的哪个**具体瞬间**？

**好的瞬间（可画）**：
- "两个人下棋，一个人偷偷把对方的棋子拿走了"
- "妈妈在洗衣，小孩在旁边学，肥皂泡飞起来"
- "一个人踩着板凳摘枇杷，猫在下面等着"
- "下雨了，两个人挤在一把伞下，一个人肩膀湿了"
- "小孩对着墙上的影子敬礼"
- "猫咪打翻了茶壶，主人回头看"

**坏的概念（不可画）**：
- "沟通的重要性"
- "人与自然的和谐"
- "成长的烦恼"
- "团队协作"
- "幸福生活"

丰子恺从来不为抽象概念配图。他画的是：某人、在某地、做某事、有个小意外或小温馨。

### 3. 设计人物姿态

人物靠姿态传情绪：

- 歪头 = 好奇、俏皮
- 低头 = 安静、专注、沮丧
- 仰头 = 望月、发呆、想事情
- 身体前倾 = 八卦、认真、紧张
- 手托腮 = 无聊、发呆、想心事
- 双手背身后 = 悠闲、散步、老头
- 一只手挠头 = 困惑、不好意思
- 两只手举着东西 = 小心翼翼、献宝

**俏皮场景的人物设计**：
- 两个小孩打架：一个揪着另一个的衣领，两人表情都没有（靠动作传达）
- 大人被猫绊了一跤：身体后仰，手在空中乱抓
- 一个人偷偷往别人茶杯里加料：身体微躬，手指做"嘘"状

### 4. 挑选自然元素

1-3 个，不要多：

| 元素 | 感觉 |
|------|------|
| 柳枝 | 春意、温柔、垂坠 |
| 月亮 | 思念、寂静 |
| 燕子 | 春天、生机 |
| 梅花 | 冬、清高 |
| 竹子 | 清瘦、简洁 |
| 远山 | 淡远、时间 |
| 芭蕉 | 雨声、闲适 |
| 桃花 | 春色、短暂 |
| 猫 | 俏皮、日常 |
| 蜻蜓/蝴蝶 | 童趣、夏天 |

### 5. 写题字

题字是"画外之音"，像画家在画边上嘀咕了一句。不是标题，不是说明。

**俏皮型**：
- "谁偷了我的枇杷"
- "猫说不是我"
- "伞呢？"

**温暖型**：
- "茶凉了，猫睡了"
- "妈妈在厨房，灯是暖的"
- "他看完信，笑了"

**哲思型**：
- "人散后，一钩新月天如水"
- "红了樱桃绿了芭蕉"
- "山路元无雨，空翠湿人衣"

**禁止**：
- 金句/Slogan
- 英文
- 网络用语
- "如图""图为"

---

## 输出格式

```
模式：{淡彩 / 纯黑白}
情绪：{核心情绪}
瞬间：{具体的、可画的生活切片，一句话}
人物：{几人，什么姿态，什么关系}
自然元素：{1-3 个}
题字："{一句}"
构图：{竖/方/横}
```
