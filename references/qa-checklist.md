# QA Checklist

## 必过项

- [ ] 线条是毛笔质感——有粗细变化、飞白、浓淡变化，不是均匀硬线
- [ ] 背景是暖白宣纸色（`#F5F0E8`），不是纯白 `#FFFFFF`
- [ ] 人物简笔——3-6 笔，无五官
- [ ] 留白 40%-60%
- [ ] 题字是手写毛笔行书——字大小不一、笔画随意、位置不刻意对齐
- [ ] 题字不是印刷体、不是碑帖楷书、不是狂草
- [ ] 1-2 个朱红印章
- [ ] 画面是"生活瞬间"不是"概念海报"
- [ ] 没有电子产品、现代交通工具、现代建筑
- [ ] 没有 Logo、二维码、对话框

### 淡彩模式额外检查
- [ ] 色彩饱和度极低
- [ ] 淡彩是湿润笔触，边界模糊
- [ ] 不超过 3 种淡彩色

### 纯黑白模式额外检查
- [ ] 只有墨色层次——浓、中、淡、极淡四层
- [ ] 没有任何淡彩色（赭石、花青、藤黄均不出现）
- [ ] 印章保留朱红色
- [ ] 画面靠墨的浓淡和干湿区分层次

## 逐项比对

### 线条质量
- [ ] 有飞白吗？（笔锋分叉，纸底露出）
- [ ] 有浓淡吗？（前景浓、远景淡）
- [ ] 有"不完美"吗？（偶尔歪一下、接头没对齐）

### 题字质量
- [ ] 像"随手写的"还是"书法作品"？
- [ ] 字大小不齐？
- [ ] 位置有点歪？
- [ ] 墨色和画面一致？

### 情绪传达
- [ ] 画面是"瞬间"还是"概念"？
- [ ] 3 秒能感受到情绪？
- [ ] 有没有俏皮感？（丰子恺不应该太严肃）

## 失败信号

### 风格失败
- 数字硬笔线条（均匀粗细、无飞白）
- 纯白背景
- 印刷体题字（致命）
- 碑帖楷书题字（太工整）
- 矢量扁平风
- 3D 渲染感
- 写实五官

### 色彩失败
- 饱和鲜艳的颜色
- 纯黑白模式出现了彩色
- 淡彩色超过 3 种

### 内容失败
- 画概念不画瞬间
- 太严肃端庄（不像丰子恺）
- 出现现代物品
- 题字太正式或太潦草

### 修复指南

| 问题 | 修复方向 |
|------|----------|
| 线条太硬 | 加重"visible brush hair splits""flying white""uneven thick-thin from hand speed" |
| 题字像印刷 | 加重"casual running script""irregular sizes""slightly tilted""NOT formal calligraphy" |
| 太严肃 | 检查场景是否是"生活瞬间"而非抽象概念；加入意外/俏皮元素 |
| 背景太白 | 加重"warm off-white xuan paper #F5F0E8""NOT pure white" |
| 缺飞白 | 加重"flying white strokes where brush splits and paper shows through" |
| 缺浓淡 | 加重"dark ink for foreground, pale ink for distance""ink density variation" |
