# 风格 DNA

## 核心理解

丰子恺称自己的画为"随笔漫画"——**随手画的草图，不是精雕细琢的作品。**

AI 的问题是把这张画当成"插图"来画——每样东西都画完、画满、画到位。丰子恺不会。他**画到意思到了就停**。剩下的留给纸、留给墨、留给看画的人自己想。

---

## 不是"简单"，是"没画完"

丰子恺的画不是"简化过的复杂画"，是**本来就只画了这么多**。

- 一棵树：不是"两笔简化"，是"他觉得两笔就够了"
- 一个人：不是"六笔提炼"，是"他画了六笔，觉得这个人已经在了"
- 空白：不是"省略了背景"，是"背景不重要，不画"

---

## 速写感，不是插图感

| 插图 | 速写 |
|------|------|
| 每样东西都画清楚 | 有些东西只画轮廓 |
| 颜色涂满一个区域 | 颜色只在一处扫一笔 |
| 背景有内容 | 背景是纸 |
| 线条匀称干净 | 线条有快慢、有毛边 |
| 构图经过设计 | 从某个角落开始画，画到哪算哪 |
| 看起来"完成"了 | 看起来"还可以再画，但不用了" |

---

## 颜色

- **淡彩模式**：黑墨为主，整张画只在一处有一笔极淡色
- **纯黑白模式**：黑墨 + 灰墨 + 纸色 + 红印章
- 纸是暖白的（`#F0E8D8`），纸的颜色本身就是画面的一部分

---

## 测试

- 这张画看起来是"画完的插图"还是"画到一半的速写"？
- 空白是"还没来得及画"还是"故意不画"？
- 颜色是不是"几乎没注意到有色"？
